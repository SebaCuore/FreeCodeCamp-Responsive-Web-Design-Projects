Introduction

C# is a modern, object-oriented programming language developed by Microsoft.

It is widely used for building a variety of applications. Some key features include:

  • Strongly Typed: Catches errors at compile time.
  • Component Oriented: Great for scalable applications.
  • Cross-Platform: Runs on Windows, Mac, and Linux via .NET.

Official Website: C# Documentation

Getting Started

To get started with C#, you need to set up your environment using the following tools:

  • Download .NET SDK: .NET Download
  • Visual Studio IDE: Visual Studio

Once you have the necessary tools, you can create a new C# project and start writing code.

Basic Syntax

C# syntax is similar to other C-based languages, making it easy to learn for developers familiar with languages like C, C++, or Java.

Here is a simple example of a C# program:

using System;
Variables

In C#, variables are declared with a type followed by a name and an optional initial value.

int age = 25;
Data Types

C# supports various built-in data types. The most common ones are:

  • int: Used for whole numbers (e.g., 10, -5).
  • double: Used for fractional numbers (e.g., 3.14, -0.5).
  • string: Used for text (e.g., "Hello World").
  • bool: Used for logic values (true or false).
int number = 10;
 double decimalNumber = 10.5;
 string name = "John";
 bool isTrue = true;
Control Structures

C# provides various control structures such as if-else statements, switch statements, and loops (for, while, do-while).

if (age > 18) {
    Console.WriteLine("Adult");
}
Functions

Functions in C# are defined using the 'void' keyword for functions that do not return a value, or a specific return type for functions that do.

void Greet(string name) {
    Console.WriteLine("Hello, " + name);
}
